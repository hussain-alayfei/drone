# موقع «عين» — Ain Website

موقع تعريفي عربي (RTL) لمشروع «عين»: درون بمستشعر أشعة جاما وذكاء اصطناعي
يحلل التربة ويوصي بالنبات الأنسب ثم يزرعه برصاصة زراعية قابلة للتحلل.

## ما يميّزه

- **الموديل الحقيقي يعمل في المتصفح**: RandomForest المصدَّر من
  `../export_web_model.py` يصنّف قراءات الجاما محلياً في صفحة الديمو —
  واختبار Vitest يطابق مخرجاته مع sklearn حتى 9 خانات عشرية (32 اختباراً).
- **خريطة سعودية تفاعلية**: 13 منطقة (حدود حقيقية من geoBoundaries) لكل
  منها توزيع جيولوجي مبرر تُسحب منه قراءة واقعية عند كل فحص.
- **مشهد 3D للرصاصة الزراعية**: procedural بالكامل (three.js بدون ملفات
  نماذج)، يتفكك ويُغرس مع السكرول، مع fallback ثابت لـ
  `prefers-reduced-motion` وأجهزة بدون WebGL.
- **هوية بصرية غير قالبية**: الألوان والخطوط وعناصر التوقيع كلها من هوية
  العرض التقديمي الرسمي — القواعد الملزمة في `DESIGN_SPEC.md`.

## التشغيل

```bash
npm install
npm run dev        # التطوير
npm run test       # اختبارات مطابقة الموديل Python ↔ TS
npm run build      # فحص TypeScript + بناء الإنتاج
npm run preview    # معاينة البناء
```

لإعادة توليد ملفات الموديل (`src/model/soil_rf.json` وأخواتها):

```bash
cd .. && python export_web_model.py
```

## النشر على Vercel

1. استورد الريبو في Vercel.
2. **Root Directory**: `web`
3. Framework preset: **Vite** (يُكتشف تلقائياً) — Build: `npm run build`،
   Output: `dist`.
